## About

**CosmicCoreX** is an Android open-source organization dedicated to maintaining device trees, kernel sources, and vendor trees for Xiaomi and other Android devices.

The organization focuses on providing clean and reliable sources for custom ROM development and device bring-up.

---

## Supported Devices

| Codename | Device |
|----------|---------|
| `garnet` | Redmi Note 13 Pro 5G / Poco X6 |
| `sky` | Redmi Note 12R Pro / Poco M6 PRO |
| `violet` | Redmi Note 7 Pro |

Additional devices may be added over time.

## Repository Structure

| Repository | Description |
|------------|-------------|
| Device Trees | Device-specific source trees |
| Kernel Sources | Linux kernel sources and modifications |
| Vendor Trees | Proprietary vendor blobs and configurations |
| Common Trees | Shared device resources and configurations |
